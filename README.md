# Cyclone Ditwah

## How a Single Storm Broke Sri Lanka

This repository contains the source code for the data-driven investigative story "How a Single Storm Broke Sri Lanka." The project documents the impact of Cyclone Ditwah, which made landfall on November 28, 2025, and explores the subsequent failure of national infrastructure and recovery efforts.

---

## Project Overview

The website is a scrollytelling experience that visualises the cascading failures across the island. It tracks the timeline of structural failure and the secondary crises following the storm.

### Core Data Points
* **Impact**: 2.2 million citizens impacted and 646 confirmed deaths.
* **Infrastructure**: 206 major transit routes blocked and over 200 active landslide zones.
* **Education**: 555,609 students cut off from schooling and 1,382 schools severely damaged or repurposed as shelters.
* **Agriculture**: 148,410 hectares of cultivated land ruined, impacting 70,000 smallholder farmers.

---

## Technical Implementation

* **GSAP & ScrollTrigger**: Power the "Collapse Timeline" sequence and narrative reveal animations.
* **Interactive Comparison Slider**: A custom component allowing users to witness the landscape changes in Kandy.
* **Map Carousel**: A phase-based navigation system for tracking the storm chronology from landfall to the northern exit.
* **Dynamic Metrics**: JavaScript-driven counters for high-impact statistics.
